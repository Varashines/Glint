# Glint

**Glint** is a high-performance, native macOS clipboard manager designed for power users who value speed, privacy, and a seamless desktop experience. It monitors your clipboard in real-time, allowing you to search and retrieve snippets with lightning efficiency.

---

## ✨ Features

- **🚀 Lightning Fast Search:** Instantly find anything you've copied with an optimized search engine.
- **🖼 Visual History:** High-quality thumbnails for images and intuitive icons for files, URLs, and text.
- **📌 Smart Pinning:** Keep your most-used snippets at the top of your history.
- **⌨️ Keyboard-Centric:** Full navigation support with global hotkeys (**⌥ + Space**) and quick-copy shortcuts (**⌘1-9**).
- **🎨 Native Design:** Built with Swift and SwiftUI for a fluid, lightweight experience that respects macOS system themes.
- **🔒 Privacy First:** Intelligent filtering for sensitive data and purely local storage.

## 🛠 Technical Architecture

Glint is built with modern Apple technologies to ensure deep integration with the macOS ecosystem:

- **Core Frameworks:** Written in **Swift** using **SwiftUI** for the user interface and **Combine** for reactive state management.
- **Clipboard Monitoring:**
    - Utilizes `NSPasteboard` change count tracking with a 0.5s polling interval on a background queue.
    - Decodes rich data types including `public.utf8-plain-text`, `public.url`, `public.file-url`, and standard image types.
- **Global Event Handling:** Employs the **Carbon** framework for reliable, system-wide hotkey registration (`RegisterEventHotKey`).
- **Data Persistence:**
    - Stores metadata in a local JSON format.
    - Manages a dedicated cache in the `Application Support` directory for high-quality image thumbnails and original captures.
    - Implements an automated background cleanup process to manage disk usage based on user-defined retention periods.

## 🛡 Privacy & Security

Glint is engineered with a "security-by-design" approach to protect your sensitive data:

- **Intelligent Filtering:** Automatically ignores clipboard content tagged with `ConcealedType`, `TransientType`, or `AutoGeneratedType` (e.g., from password managers).
- **Active Blocklist:** Built-in protection for industry-leading security applications:
    - **Apple:** Keychain Access, System Passwords, and the native Passwords app.
    - **Third-Party:** 1Password, Bitwarden, Dashlane, LastPass, Enpass, and KeePassXC.
    - **Hardware:** Trezor Bridge.
- **Zero Cloud:** All data is stored strictly on your local machine. No telemetry or clipboard data is ever transmitted to external servers.

## 🏗 Development

### Prerequisites
- macOS 13.0 or later
- Xcode 14.0+ / Swift 5.7+
- [dmgbuild](https://github.com/dmgbuild/dmgbuild) (optional, for local DMG packaging)

### Building Locally
Clone the repository and build using Swift Package Manager:

```bash
git clone https://github.com/Varashines/Glint.git
cd Glint
swift build
```

### Running the App
```bash
swift run
```

## 📦 Distribution & Releases

Glint uses an automated CI/CD pipeline via **GitHub Actions** for consistent build delivery.

### Automated Releases (CI)
Official releases are triggered via Git tags:
1. Create an annotated tag: `git tag -a v1.x.x -m "Release notes"`
2. Push the tag: `git push origin v1.x.x`

The CI pipeline automatically generates a Universal Binary, builds the icon set, and packages a notarizable DMG.

### Manual DMG Creation
To generate a professional DMG locally:

```bash
pip install dmgbuild
cd Glint
swift PackageApp.swift
```
